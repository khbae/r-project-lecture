문제5
  • 원주율을 구하는 공식은 다음과 같다.
  • 아래 공식과 for문을 활용하여 원주율을 계산하는 함수를 만드시오

x = 1000000
pi = 0
for(i in 1:x){
   if(i%%2 == 1){
      pi = pi + 4/(2*i-1)
   }
   else{
      pi = pi - 4/(2*i-1)
   }
}
pi
## [1] 3.141592

    for(i in 1:nchar(x)){
